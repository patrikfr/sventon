Release road map

This is a rough release plan for the sventon, it will be updated and changed 
irregularily.

[ ] = planned
[x] = in progress
[<rev>] = completed in revision

Release 1
[<76>] Browse WebDAV and svnserve repositories (anonymous and authorized login).
[<76>] View file for any given revision.
[<76>] Source code syntax highlighting.
[<76>] View logs for a file/directory.
[ ] View all changes in one revision (log entry).
[x] Download file (any given revision).
[<76>] Browse repository for any given revision.
[<76>] Possibility to navigate in repository by typing path.
[<76>] Clickable revision path info for quick navigation in browser.
[x] Basic look and feel
[ ] Support for language specific characters and white space in paths (WebDAV)
[x] Possibility to search for files and directories.
[x] Blame
[x] Flatten path structure view
[x] Full style sheet usage to enable easy customization

Not scheduled
[ ] Possibility to configure a view only user + pwd to avoid end user authentication
[ ] Serialize the RevisionIndex to disk
[ ] Only append repository changes if index is out-of-date instead of full reindexing
[ ] Apply Observer pattern and make RevisionIndex Observable
[ ] RSS feeds of repository changes, RevisionIndexObserver
[ ] Commit statistics (different kinds)
[ ] Diff revisions of same file
[ ] Diff same revision of different files
[ ] View contents of zip archives (including jar)
[ ] Download file(s) and directories as zip files
[ ] JSPWiki integration
[ ] Keyword expansion when viewing files
[ ] User bookmark support using persistent cookies
[ ] View metadata (e.g. mime types)
[ ] Installation wizard
[ ] Subversion 1.2 feature support, notably file locking.
[ ] Integration with other tools and IDE:s
[ ] Content search using lucene (files in HEAD)
[ ] Graph logs
